#Student Management System
Overview

The Student Management System is a console-based application developed in C++ that demonstrates the use of Object-Oriented Programming (OOP) concepts and the Standard Template Library (STL). The application provides an efficient way to manage student records through a simple menu-driven interface.

#Features\
Add new student records\
Automatically generate unique student IDs\
View student details\
Update student names and marks\
Delete student records\
Search students by ID\
Search students by name (supports partial matches)\
Calculate individual student average marks\
Calculate class average marks\
Display total number of students\
Faculty module for viewing student performance\
